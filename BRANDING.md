# Peak Perfection – Logo & Favicon

## Favicon

- The app uses **`/logo.png`** as the favicon (same file as the header logo).
- **Steps:** Place your logo at **`stylist-app/public/logo.png`**. The browser tab and dashboard header will both use it.
- **Page title:** Set in `index.html` as **"Peak Perfection Stylist OS"**.

## Logo (dashboard header and favicon)

- **Where to add:** Place your logo in the **`public`** folder as **`logo.png`** (`stylist-app/public/logo.png`).
- **Dashboard:** Shown in the top left at ~40px height, next to the text "Peak Perfection". Clean, minimal internal-software layout.
- **Favicon:** The same `logo.png` is used as the browser tab icon (set in `index.html`).
- If the file is missing, only the text "Peak Perfection" is shown in the header (no broken image).
